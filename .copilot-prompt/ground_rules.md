
# あなたが作業をする上で最も大事にするべきルール定義です

## 全体方針

1. github copilot、および Agent による返答は全て日本語で返答してください。
2. Agent mode により、コーディング及びファイル編集をする場合は必ず、私へ確認を行ってから編集などを行ってください。

## Copilot としてコーディング、調査を行う場合、作業の記録を以下のルールに従い行ってください

1. ファイルを生成し、自動生成したものは常にファイル冒頭に次のコメントをつけてください: 'Generated by Copilot'
2. Agent modeとして確認、および実行した内容を './tmp/copilot-logs/<今のブランチ名>-work-copilot.md' という作業ファイルとして記録するようにしてください。
  a. git 管理されてないフォルダによって利用されている場合は './tmp/copilot-logs/work-copilot.md' を作業ファイル名にしてください。
3. 新たに作業を開始する場合は 作業ファイル (つまり'./tmp/copilot-logs/<今のブランチ名>-work-copilot.md') がある場合はそれを復元するかどうか必ず尋ねるようにしてください。
4. 「作業をSaveして」と指示があった場合は速やかに 作業ファイルの内容を更新し、これまでの指示によって変更があった内容が正しく作業ファイルに更新されているか確認し、更新してください。
5. 作業ファイルの更新時は、必ず変更内容を diff形式で 指示者に確認してください。
